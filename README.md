# Pac-Man
This is a copy of Pac-Man Game created using HTML, CSS and Vanilla JavaScript.
# How To Play
Use keys 'w', 'a', 's', and 'd' to operate the Pac-Man up, left, down, and right. (Caps Lock Off)
